A simple and clean **React.js application** to manage internship-related tasks.  
This project is designed for beginners to demonstrate core React concepts in a practical way.

---

##  Features

- Add new internship tasks
- View all tasks in a clean UI
- Mark tasks as **Completed / Pending**
- Delete tasks
- Data stored using **Local Storage** (no backend required)

---

##  Tech Stack

- React.js (Vite)
- JavaScript (ES6)
- CSS (Custom styling)
- React Router DOM

---

##  Pages

1. **Task List Page**
   - Displays all tasks
   - Update task status
   - Delete tasks

2. **Add Task Page**
   - Add new internship tasks with title & description

---

##  Concepts Used

- Functional Components
- React Hooks (`useState`, `useEffect`)
- React Router
- Local Storage
- Conditional Rendering
- Basic CRUD Operations
